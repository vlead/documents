2 Requirements

2.1 Resource Requirements

Number of VMS
About 140 labs and other services, therefore about 150 VMS
Containers or VMs
VM per each lab or a large VM containing multiple containers.
Varying Specifications
Each of the VM might vary from 2 GB RAM to 16 GB RAM.

2.2 Elasticity

Process intensive labs
Would need more CPU and probably more RAM than labs that serve static pages.
Popularity of a lab
More resource allocation on demand.
Cannot exactly predict resource requirement
Rely on scalability and dynamic resource allocation of cloud.

2.3 API Requirements of Cloud

Management of VMs
Creation, termination, reboot etc through an API.
Necissity of API
Management of Labs (deployment, undeployment, resource usage) with manual intervention.
External Resource Monitoring
Cloud services to monitor VMs - to delink the dependency on a local agent
